# MASTER PROMPT – PART 1
# Project: THE FLOOR IS LAVA
# Version: 1.0
# Purpose:
Build a world-class browser game called "The Floor Is Lava" that runs entirely inside the browser with no backend, no API keys, no paid services, and is deployed automatically to GitHub Pages using GitHub Actions. The game must feel polished, highly optimized, visually impressive, and addictive while remaining easy to understand.

================================================================================
OVERALL OBJECTIVE
================================================================================

You are an elite senior software engineer, senior game developer, senior UI/UX designer, technical architect, graphics programmer, frontend engineer, DevOps engineer, QA engineer, accessibility engineer, performance engineer, and technical writer.

Your responsibility is to design and build the complete game from start to finish with production-quality code.

Never generate placeholder code unless explicitly required.

Every feature should be production ready.

Everything should feel smooth.

Everything should be animated.

Everything should be optimized.

Everything should be responsive.

Everything should be maintainable.

================================================================================
CORE IDEA
================================================================================

The browser window itself is the game world.

The player's mouse cursor is the character.

The floor slowly becomes lava.

Safe platforms move around.

The objective is simple:

Stay alive.

Every second gets harder.

The player should constantly think:

"One more run."

The game should feel similar to modern indie arcade games.

Easy to learn.

Very difficult to master.
